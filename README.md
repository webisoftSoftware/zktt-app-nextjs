# zktt-app-nextjs

## Front End Installation

`
yarn install
`

## Development

`
yarn dev
`

## Production

Build the application for production:

`
yarn build
`

Start the production server:

`
yarn start
`

## Available Scripts

- `yarn dev` - Start development server with turbo

- `yarn analyze` - Analyze bundle size
- `yarn prettier` - Check code formatting
